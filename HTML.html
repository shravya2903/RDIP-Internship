HTML

HTML stands for Hyper Text Markup Language. It is a standard markup for creating web pages. This page describes a bref understanding of HTML.

HTML describes the structure of a web page and consists of a series of elements. These elements tell the browser how to display the content.

Some of the elements in HTML are as follows:

  • The !doctype html tag defines that the document is an HTML5 document.
  • the html tag is the root element of an HTML page.
  • the head tag contains the header, also called the meta information about the HTML page, for exmaple, the title of the page.
  • the body element defines the body of the document and is also a container for all the visible contents, such as the headings.
  • the heeading element denoted from h1 to h6 defines headings of various sizes from largest being the h1 to the smallest being the h6.
  • the paragraph element, denoted by p, defines a paragraph.

All the elemeents in HTML are defined by a start tag, some content and an end tag.
